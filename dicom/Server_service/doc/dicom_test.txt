для подключения к паксу/ворклисту
мне нужна по аппаратам где не поддерживается DICOM
информация (лучше сразу все собрать в таблицу - т.к. все понадобится) :
- название 
- модальность - тип аппарата
- модель
- серийный номер (можно бирку на аппарате сфотографировать) - серийник нужен если придется общаться с поддержкой производителя
- операционная система на компьютере который управляет аппаратом и на рабочем месте врача
- поддержка worklist (наличие рабочей опции которую не надо покупать)
- возможность установить программу видеозахвата если dicom не поддерживается
- посмотреть есть ли у аппарата подключение к сети
- посмотреть есть ли у аппарата возвожность записи на dicom диск (CD/DVD)? 
- есть ли сохранение на флеш-диск

по аппаратам с поддержкой dicom - узнать есть ли поддержка worklist (посмотреть есть ли доступные настройки и там должна быть кнопка echo)


например на узи где стоит windows и нет dicom но есть сеть можно поставить программу видеозахвата
